## 📋 Roadmap Overview
 
This repository tracks a comprehensive, **200-day de-duplicated DSA mastery plan**. All daily solution codes are written in optimized **C++** and organized systematically into **9 distinct learning phases** under their respective topic folders.
 
*   **Daily Routine:** 2 New LeetCode Problems + 1 Revision LeetCode Problem (for spaced recall).
*   **Total Problems:** 400 100% unique LeetCode problems covering core data structures, advanced algorithms, and final mock revision sprints.
*   **Style Guide:** Solutions contain detailed complexity analysis ($O(N)$ time, $O(1)$ space, etc.) and brief algorithmic intuitions.
